## 设计文档写作约定

- DDL 中所有业务表必须包含公共字段（id / tenant_id / created_by / created_at / updated_by / updated_at / deleted）
- 接口路径统一前缀 `/api`，权限说明写在接口旁
- 与 scp-foundation 的差异点在每份文档末尾统一列出，便于开发时快速辨别坑点
